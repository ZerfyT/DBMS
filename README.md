# DBMS
#### A simple Database Management System build with Java Swing and MySQL server.

## Available Features

|Name|Availability|
|----|------------|
|Create new database|Yes|
|Delete database|Yes|
|Create new table|Yes|
|Datatypes - Varchar, Int, Date|Yes|
|Update table|-|
|Update table data|-|
|Delete single tuple/row|Yes|
|Delete all data in table|Yes|
|Delete table|Yes|
|Primary Key|Yes|
|Foreign Key|-|
